# vue-todos

This template should help get you started developing with Vue 3 in Vite.

## Clone & Download

$ git clone https://github.com/aminase/vue-todos.git <br>
$ cd vue-todos <br>
$ npm install <br>
$ npm run dev<br>

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
